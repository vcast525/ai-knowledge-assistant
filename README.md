# AI Knowledge Assistant

## Overview

AI Knowledge Assistant is a full-stack software engineering application designed to help users upload, search, and retrieve information from documents through an intuitive web interface.

The application supports document ingestion, knowledge retrieval, search capabilities, and AI-assisted question answering.

## Supported File Types

- PDF
- DOCX
- TXT
